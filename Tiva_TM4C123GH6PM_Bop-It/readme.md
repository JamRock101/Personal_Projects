Designed and programmed a single-player “Bop-It” RTOS-based game on a Tiva TM4C123GH6PM, integrating buttons, joystick, accelerometer, LCD display, and speaker for interactive gameplay.
Implemented real-time scoring and input detection in C using a custom real-time operating system for responsive gameplay logic.
